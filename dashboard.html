:: DESIGN_DASHBOARD

How to use: Since this site is hosted on GitHub Pages (Static), you cannot upload files directly to a server.

  1. Upload your image to the assets/designs/ folder in your repository (or use an external URL).
  2. Fill out the form below.
  3. Copy the generated JSON code.
  4. Paste it into your designs.json file.
  5. Commit and push your changes.
